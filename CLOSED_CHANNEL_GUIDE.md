## 🔑 초대 코드 시스템

### 작동 방식

```
관리자가 채널 생성
        ↓
자동으로 6자리 초대 코드 생성 (예: ABC123)
        ↓
코드를 카톡/문자/메신저로 공유
        ↓
받는 사람이 앱에서 "초대 코드로 가입" 선택
        ↓
코드 입력 (ABC123)
        ↓
✅ 채널 가입 완료!
```

### 초대 코드 특징

- **6자리** 대문자 + 숫자
- **고유함**: 중복 없이 생성
- **혼동 방지**: I, O, 0, 1 제외
- **대소문자 자동 변환**: 소문자 입력해도 자동으로 대문자로
- **관리자만 확인 가능**: 일반 멤버는 볼 수 없음
- **재생성 가능**: 관리자가 필요 시 새 코드 생성 가능

### 예시 초대 코드
```
✅ ABC123
✅ XYZ789
✅ QWE456
✅ RST234
✅ MNP567
```

## 📱 사용 시나리오

### 시나리오 1: 팀 채널 만들기

**관리자 (팀장)**
1. 앱 실행 → 채널 목록
2. "+ 새 채널" 클릭
3. 채널 이름: "프로젝트 팀"
4. 채널 생성 → 초대 코드 확인: **ABC123**
5. 팀원들에게 공유:
   ```
   안녕하세요!
   세이투두 긴급 알림 채널에 가입해주세요.
   
   초대 코드: ABC123
   
   앱 설치 후 "초대 코드로 가입"에서 입력하시면 됩니다.
   ```

**팀원 (멤버)**
1. 앱 설치 및 구글 로그인
2. 채널 목록 화면
3. "🔑 초대 코드로 가입" 클릭
4. **ABC123** 입력
5. ✅ "프로젝트 팀" 채널 가입 완료!

### 시나리오 2: 가족 채널

**관리자 (부모님)**
1. 채널 생성: "우리 가족"
2. 초대 코드: **XYZ789**
3. 가족 단톡방에 공유

**가족 구성원**
1. 앱에서 **XYZ789** 입력
2. 즉시 가입!

### 시나리오 3: 긴급 알림 발송

**관리자**
1. 채널 상세 → "📢 긴급 알림 발송"
2. 제목 입력: "긴급 회의"
3. 메시지 입력: "10분 후 회의실 집합"
4. 발송! 📞

**모든 멤버**
- 전화벨처럼 울림! 🔔
- 화면 자동 켜짐
- 수락/거절 선택

## 🔐 개인정보 보호

### 왜 폐쇄형인가?

1. **프라이버시**: 내 채널을 다른 사람이 찾을 수 없음
2. **보안**: 초대받은 사람만 가입 가능
3. **스팸 방지**: 무분별한 가입 차단
4. **관리 용이**: 내가 초대한 사람만 관리

### 멤버 초대 방법 (2가지만)

#### 방법 1: 초대 코드 (추천! ⭐)
- **장점**: 간단하고 빠름
- **단점**: 코드를 아는 사람은 누구나 가입 가능
- **용도**: 팀, 그룹, 커뮤니티

#### 방법 2: 이메일 초대
- **장점**: 특정 사람만 정확히 초대
- **단점**: 이메일을 정확히 알아야 함
- **용도**: 개인적인 초대, VIP 멤버

## 💡 주의사항

### ⚠️ 초대 코드 관리
- 초대 코드는 **관리자만** 볼 수 있습니다
- 코드가 유출되면 누구나 가입 가능합니다
- 필요 시 **코드 재생성**으로 기존 코드 무효화

### ⚠️ 채널 검색 불가
- 앱 내에서 채널을 검색할 수 없습니다
- 초대 코드나 이메일로만 가입 가능합니다
- 이것은 **의도된 설계**입니다!

### ⚠️ 멤버 관리
- 각 채널은 **독립적**입니다
- 다른 채널의 멤버를 볼 수 없습니다
- 관리자는 자기 채널만 관리합니다
