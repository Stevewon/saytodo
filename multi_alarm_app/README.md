# 🚨 멀티 알람 앱

1대 기기에서 여러 알람을 동시에 관리하는 Flutter 앱

---

## ✅ 구현 완료

### 화면 (2개)
- ✅ **홈 화면**: 알람 목록 + 활성/비활성 개수
- ✅ **알람 추가 화면**: 시간, 제목, 반복 요일, 진동 설정

### 기능 (10개)
- ✅ **여러 알람 추가**: 무제한 알람
- ✅ **알람 ON/OFF**: 스위치로 간편하게
- ✅ **반복 설정**: 월~일 요일별 반복
- ✅ **빠른 설정**: 평일/주말/매일 버튼
- ✅ **알람 삭제**: 개별 삭제
- ✅ **알람 복제**: 비슷한 알람 빠르게 생성
- ✅ **남은 시간 표시**: 다음 알람까지 시간
- ✅ **진동 설정**: 알람별 진동 ON/OFF
- ✅ **정렬**: 시간순 자동 정렬
- ✅ **로컬 저장**: SharedPreferences

## 🎯 주요 기능

### 1️⃣ 여러 알람 동시 관리
```dart
// 무제한 알람 추가 가능
AlarmModel(
  title: '기상 알람',
  time: DateTime(2024, 1, 1, 7, 0),
  repeatDays: [0, 1, 2, 3, 4], // 평일
)
```

### 2️⃣ 반복 요일 설정
- **평일**: 월~금
- **주말**: 토~일
- **매일**: 월~일
- **커스텀**: 원하는 요일만 선택

### 3️⃣ 알람 ON/OFF
- 스위치로 간편하게 켜고 끄기
- 비활성 알람은 회색으로 표시
- 활성 알람만 개수 카운트

### 4️⃣ 남은 시간 표시
```
"3시간 24분 후"
"8시간 후"
"2일 5시간 후"
```

### 5️⃣ 알람 복제
- 비슷한 알람을 빠르게 생성
- 시간, 반복, 설정 모두 복사
- 제목에 "(복사)" 자동 추가
