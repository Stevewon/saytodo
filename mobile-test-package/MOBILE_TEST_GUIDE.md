## 🎬 모바일 테스트 시나리오

### 1. 로그인 테스트
```
1. 링크 열기
2. "Google로 계속하기" 버튼 터치
3. 2초 대기
4. ✅ 채널 목록 화면 진입
```

### 2. 채널 생성 테스트
```
1. 우측 하단 "채널 만들기" FAB 터치
2. 채널 이름 입력: "테스트 채널"
3. 설명 입력: "모바일 테스트"
4. "채널 만들기" 버튼 터치
5. ✅ 채널 생성 완료!
```

### 3. 초대 코드 테스트
```
1. 채널 카드 터치
2. 상단 [공유] 아이콘 터치
3. ✅ 6자리 초대 코드 확인 (예: FAM123)
4. "코드 복사" 버튼 터치
5. ✅ 클립보드에 복사됨
```

### 4. 음성 메시지 테스트
```
1. 채널 상세 화면
2. 하단 [🎤 음성] 버튼 터치
3. "녹음 시작" 버튼 터치
4. 3초 대기 (시뮬레이션)
5. "녹음 중지" 버튼 터치
6. "전송하기 (3명)" 버튼 터치
7. ✅ 전송 완료 알림!
```

### 5. 영상 전송 테스트
```
1. 채널 상세 화면
2. 하단 [🎥 영상] 버튼 터치
3. "녹화 시작" 버튼 터치
4. 5초 대기
5. "녹화 중지" 버튼 터치
6. "전송하기 (3명)" 버튼 터치
7. ✅ 전송 완료!
```

### 6. 유튜브 링크 테스트
```
1. 채널 상세 화면
2. 하단 [🔗 링크] 버튼 터치
3. 유튜브 URL 입력
4. "전송하기 (3명)" 버튼 터치
5. ✅ 링크 공유 완료!
```
