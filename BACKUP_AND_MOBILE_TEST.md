## 🎬 모바일 테스트 시나리오

### ✅ 기본 흐름
```
1. 링크 열기 → 로그인 화면
2. "Google로 계속하기" 터치
3. 2초 대기 → 채널 목록
4. 샘플 채널 2개 확인
5. "채널 만들기" FAB 터치
6. 채널 정보 입력 후 생성
7. 채널 카드 터치 → 상세 화면
8. [공유] 버튼 → 초대 코드 확인
9. [🎤 음성] → 녹음 → 전송
10. ✅ 완료!
```
